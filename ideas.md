# Brainstorm: Diseño de Landing Page — Ascente CFO®

## Contexto
Landing page para el servicio "Diagnóstico de Claridad Financiera®" de Ascente CFO®. Público objetivo: directores generales y propietarios de PYMEs españolas. Objetivo principal: conversión hacia el formulario de contratación en Tally. Color corporativo: #1C00A8 (azul oscuro) y #2800ff (azul vibrante).

## Idea 1: "Corporate Gravity" — Peso Visual Institucional

**Movimiento de Diseño:** Inspirado en el Brutalismo Corporativo de firmas como McKinsey y Goldman Sachs, donde la autoridad se transmite a través de la densidad tipográfica y la ausencia de decoración innecesaria.

**Principios Fundamentales:**
- Tipografía como arquitectura: los títulos ocupan el 60% del viewport, creando un efecto de "muro de texto" que transmite seriedad
- Contraste extremo entre secciones (azul oscuro #1C00A8 alternando con blanco puro)
- Datos numéricos como protagonistas absolutos, con tamaños de fuente de 120px+

**Filosofía de Color:** El azul #1C00A8 se usa como color dominante (70%), representando la profundidad del análisis financiero. El blanco actúa como "respiro" entre secciones densas. Un rojo (#E53935) aparece exclusivamente en los datos de alerta.

**Paradigma de Layout:** Secciones de ancho completo que alternan entre fondos oscuros y claros. Cada sección ocupa el 100vh. Los datos estadísticos se presentan en formato de "ticker financiero" horizontal.

**Elementos Distintivos:**
- Números gigantes (42%, 46%) que aparecen como watermarks detrás del texto
- Líneas horizontales gruesas que dividen secciones como en un informe financiero impreso
- Tabla de precios con estilo de "terminal Bloomberg"

**Filosofía de Interacción:** Scroll suave entre secciones con efecto de "peso" — cada sección se detiene brevemente antes de continuar, simulando la gravedad de la información.

**Animación:** Números que cuentan desde 0 hasta su valor final al entrar en viewport. Secciones que se deslizan con inercia pesada.

**Sistema Tipográfico:** Space Grotesk (800) para títulos monumentales, Inter (400) para cuerpo. Contraste extremo de pesos.

## Idea 2: "Financial Clarity Blueprint" — Precisión Analítica

**Movimiento de Diseño:** Inspirado en el diseño de dashboards financieros de alta gama (Bloomberg Terminal meets Swiss Design), donde la información se presenta con precisión quirúrgica y jerarquía visual impecable.

**Principios Fundamentales:**
- Grid de 12 columnas visible como guía sutil en el fondo
- Semáforos de color (verde/ámbar/rojo) como sistema de señalización visual
- Micro-tipografía para datos técnicos, macro-tipografía para mensajes clave

**Filosofía de Color:** El azul #1C00A8 como base de autoridad. El #2800ff como acento para CTAs y elementos interactivos. Gris grafito (#2D2D2D) para texto secundario. Fondo blanco con textura sutil de papel financiero.

**Paradigma de Layout:** Layout asimétrico con columna principal (8 cols) y columna de datos (4 cols) que actúa como "sidebar de ratios". Las secciones se organizan como un informe financiero real: portada, resumen ejecutivo, análisis, conclusiones.

**Elementos Distintivos:**
- Barra lateral fija con indicadores de progreso que muestra en qué sección estás
- Cards con bordes izquierdos coloreados según semáforo (verde/ámbar/rojo)
- Iconografía de línea fina estilo "blueprint" para los 5 bloques de análisis

**Filosofía de Interacción:** Navegación por secciones con indicador lateral. Los datos se revelan progresivamente al hacer scroll, como si se estuviera "desclasificando" un informe.

**Animación:** Elementos que aparecen con fade-in sutil y desplazamiento de 20px desde abajo. Números que se "escriben" como en una calculadora financiera. Transiciones de 300ms con ease-out.

**Sistema Tipográfico:** DM Sans (700) para títulos, Inter (400/500) para cuerpo y datos. Monospace (JetBrains Mono) para cifras y ratios financieros.

## Idea 3: "Ascente Noir" — Dramatismo Financiero

**Movimiento de Diseño:** Inspirado en el cine noir y las presentaciones de startups de Silicon Valley (Stripe, Linear), donde el fondo oscuro crea un escenario dramático para la información y los CTAs brillan como neón.

**Principios Fundamentales:**
- Fondo predominantemente oscuro (#0A0A1A) con el azul corporativo como luz de acento
- Gradientes sutiles que simulan profundidad y dimensión
- Espaciado generoso que crea tensión visual entre elementos

**Filosofía de Color:** El negro profundo (#0A0A1A) como lienzo. El azul #2800ff como "luz" que ilumina los elementos clave. Gradiente de #1C00A8 a #2800ff para CTAs y highlights. Blanco (#F5F5F5) para texto sobre fondos oscuros.

**Paradigma de Layout:** Secciones de ancho completo con contenido centrado en max-width de 1200px. Hero section con efecto de "spotlight" sobre el título. Las secciones de datos usan grids de 3 columnas con cards flotantes sobre el fondo oscuro.

**Elementos Distintivos:**
- Efecto de "glow" azul detrás de los elementos clave (como un halo de luz)
- Líneas de gradiente que conectan secciones visualmente
- Botón CTA con efecto de "pulso" luminoso que atrae la atención

**Filosofía de Interacción:** Hover effects con glow sutil. Los elementos se iluminan al pasar el cursor, como si se activaran. El scroll revela contenido con efecto de "fade from darkness".

**Animación:** Parallax sutil en el hero. Elementos que emergen de la oscuridad con blur-to-focus. Glow pulsante en CTAs cada 3 segundos.

**Sistema Tipográfico:** Sora (700/800) para títulos con tracking amplio, Inter (400) para cuerpo. El contraste entre la geometría de Sora y la neutralidad de Inter crea tensión visual.

## Decisión Final

Selecciono la **Idea 2: "Financial Clarity Blueprint"** por las siguientes razones:

1. **Coherencia con la marca:** El estilo de dashboard financiero refuerza la credibilidad de Ascente CFO como servicio de CFO externo profesional.
2. **Conversión:** El layout asimétrico con sidebar de navegación guía al usuario a través del contenido de forma estructurada, aumentando el tiempo en página.
3. **Legibilidad:** El fondo claro con acentos de color facilita la lectura de datos financieros complejos.
4. **Diferenciación:** Se aleja del diseño genérico de landing pages SaaS y se posiciona como una herramienta financiera seria.
5. **Accesibilidad:** El contraste alto entre texto oscuro y fondo claro cumple con WCAG AA.
